Web Developer Interview Questions

HTML

1. What is the purpose of the <!DOCTYPE> declaration in HTML?

The <!DOCTYPE> declaration in HTML specifies the document type and version, ensuring proper rendering of the HTML document by the browser.

2. Explain the difference between <div> and <span> elements.

The <div> element is a block-level container, while the <span> element is an inline container. They are used to group and style content on a web page.

CSS

3. Explain the difference between margin and padding.

Margin is the space outside an element, while padding is the space inside an element's border. They both affect the layout and spacing of elements.

4. How does the CSS box model work?

The CSS box model describes the layout of elements on a web page, consisting of content, padding, border, and margin.

JavaScript

5. Explain the difference between let, const, and var in JavaScript.

let and const are block-scoped, while var is function-scoped. const cannot be re-assigned, and let allows re-assignment.

6. What is the purpose of the this keyword in JavaScript?

this refers to the current execution context in JavaScript and is often used in object-oriented programming to refer to the current object.

React

7. What is React and what problem does it solve?

React is a JavaScript library for building user interfaces. It solves the problem of efficiently updating and rendering components in response to data changes.

8. Describe the component lifecycle in React.

The React component lifecycle consists of mounting, updating, and unmounting phases, each with specific lifecycle methods like componentDidMount and componentWillUnmount.

9. What is the difference between id and class attributes in HTML?

The id attribute is used to uniquely identify an HTML element, while the class attribute is used to assign one or more class names to an element for styling or scripting purposes.

10. How do you embed an image in HTML?

To embed an image in HTML, you use the <img> element with the src attribute specifying the image file's URL.

11. How does the CSS z-index property work?

The z-index property in CSS controls the stacking order of positioned elements. Elements with a higher z-index value appear above those with a lower value.

12. How do you center an element horizontally and vertically in CSS?

To center an element horizontally, you can use margin: auto;. To center vertically, you might use display: flex; and align-items: center; or other techniques.

13. What is the significance of the JSON.stringify and JSON.parse methods?

JSON.stringify converts a JavaScript object to a JSON string, and JSON.parse converts a JSON string back to a JavaScript object.

14. Explain the concept of closures in JavaScript.

Closures in JavaScript allow functions to retain access to variables from their outer (enclosing) scope even after the outer function has finished executing.

15. What is the difference between functional components and class components in React?

Functional components are simpler and stateless, while class components can hold and manage state using the React lifecycle methods.

16. How does React Router work, and why is it used?

React Router is a library for routing in React applications. It allows you to define navigation paths and view different components based on the URL, creating a single-page application experience.
